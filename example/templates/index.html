Muffin Pastry Shop

Hello ! Welcome to Muffin, Kopo Kopo's sample application for integrating with our python library. We are happy to have you here.
Muffin is a pastry shop that uses this sample application to manage its payments to suppliers, receive payments from customers, track financial transactions on the Kopo Kopo app and settle their money.
In this sample application you will be able to see all of Kopo Kopo's API services as would be run in a sandbox environment. To see examples of these implementations of the python library please navigate using the navigation panel on the left side.
